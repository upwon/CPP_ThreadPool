# CPP_ThreadPool
 Simple C++ Thread Pool   

一个简单的C++线程池示例   

  <br/>

  <br/>

Usage

使用方法

```bash
git clone https://github.com/upwon/CPP_ThreadPool.git  
cd CPP_ThreadPool  
mkdir build  
cd build  
cmake ../  
make  
./CPP_ThreadPool   

```
